Exercise: 

                            
Add a "tooltip" to the paragraph below with the text "About W3Schools".

                            

                                
<p="About W3Schools">Codour is a web developer's site.</p>

Exercise 3
                            
Exercise: 

                            
Make the element below into a link that goes to "https://www.codour.com".

                            

                                
<a  = "https://www.codour.com">This is a link</a>
Exercise 4
                            
Exercise: 

                            
Specify an alternate text for the image.

                            
Alternate text is useful when the image cannot be displayed, like when the page is read by a screen reader.

                            

                                
<img src="codour.png" ="codour Logo">
